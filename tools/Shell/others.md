# Tools


### Tmux

add custom settings

```
echo set-option -g default-shell /usr/bin/fish > ~/.tmux.conf
```

primero control+b suelta y presiona el siguiente comando

`C-b  %` => División Vertical
`C-b  "`  => División Horizontal
`C-b  [Flechas]` =>  Navegacion
